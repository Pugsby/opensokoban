# OpenSokoban
A open source version of Sokoban.
## Controls
- Move: Arrow Keys
- Undo: Ctrl+Z or Backspace
- Change Level: L
## Build Instructions
```
git clone https://github.com/Pugsby/opensokoban
cd opensokoban
cargo build --release
```
The binary/program will be in `./target/release/opensokoban`. You can either run it directly or move it to a folder included in your PATH.
